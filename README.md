# FE-528 — Technician Work Order Usage Tickets Harness

Standalone Angular 18 harness app for the Technician Work Order Usage Tickets feature. Uses mock data only — no API calls, no backend required.

## Quick Start

```bash
cd fe-harness-FE-528
npm install
ng serve
```

Open `http://localhost:4200`. The Work Order Details page loads by default.

## What This Is

This harness starts from the Work Order Details page (carried over from FE-6523) and adds a new page for Technician Work Order Usage Tickets that launches from it.

## Tech Stack

- Angular 18.2
- TypeScript 5.5
- `@assetworks-llc/aw-component-lib@26.1.1-ng18`
- SCSS
